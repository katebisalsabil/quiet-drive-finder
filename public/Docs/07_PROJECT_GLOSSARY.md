# Project Glossary

## API
A service your app talks to in order to get information, such as route or traffic data.

## API Key
A private code that lets your app use an API. Do not share it publicly.

## Commit
A saved checkpoint in Git.

## Git
A tool that saves versions of your code.

## GitHub
A website where you store your Git project online.

## Leaflet
A JavaScript library for showing interactive maps.

## Localhost
Your computer running your app locally, usually at a URL like `http://localhost:5173`.

## npm
A tool that installs JavaScript packages.

## React
A JavaScript library for building web app interfaces.

## Repository / Repo
A project folder saved with Git/GitHub.

## Route
A path from one location to another.

## Vite
A tool that creates and runs modern React apps quickly.

## VS Code
The code editor where you write your project.
